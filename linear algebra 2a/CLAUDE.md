# אלגברה לינארית 2 — פרויקט קורס אינטראקטיבי

## מה הפרויקט הזה

בניית קורס לימוד אינטראקטיבי לאלגברה לינארית 2 (TAU) מבוסס על:
- **הרצאות יהודה שלום (2019)** — 87 עמודים, עמוד הגב התיאורטי של הקורס
- **תרגולים** — מגיעים לאורך הסמסטר; מכילים הגדרות + דוגמאות מלאות עם פתרונות
- **תרגילי בית** — קובעים את רמת הקושי; שאלות הוכחה + חישוב ברמה גבוהה

## מבנה ה-vault

```
vault/
  00-MOC.md                ← מפת תוכן ראשית — נווט מכאן
  _templates/              ← תבניות callout לשימוש חוזר
  _settings/               ← הגדרות Obsidian ורשימת תוספים
  ch01-eigenvalues/
    _index.md              ← דף ניווט לפרק
    01-lecture.md          ← תיאוריה מהרצאות (הגדרות, משפטים, הוכחות)
    02-tirgul.md           ← דוגמאות מהתרגול עם פתרונות מלאים
    03-exercises.md        ← תרגילים ברמת שיעורי בית, עם פתרונות מרובדים
    04-flashcards.md       ← כרטיסיות לחזרה מרווחת
  ... (פרקים נוספים באותו מבנה)
```

## כללים לעבודה

### שפה
- **כל התוכן בעברית** — הרצאות, תרגולים, תרגילים, רמזים, פתרונות
- קיצורים סטנדרטיים: ע"ע (ערך עצמי), ו"ע (וקטור עצמי), ג"ר (ריבוי גיאומטרי), ר"א (ריבוי אלגברי), ל"ט (העתקה לינארית), מ"מ (מרחב וקטורי)

### מבנה שאלות — שלושה שלבי רמז
כל תרגיל הוכחה מכיל שלושה callouts מקוננים:
```markdown
> [!question]- שאלה: [כותרת]
> [טקסט השאלה]
>
> > [!tip]- רמז
> > [רמז כיווני — לא מגלה את הפתרון]
> >
> > > [!example]- פתרון חלקי
> > > [הצעד הקריטי — מה לעשות אבל לא הכל]
> > >
> > > > [!success]- פתרון מלא
> > > > [הוכחה מלאה עם כל הפרטים]
```

### תגיות חובה בכל תרגיל
- מקור: `#lecture` / `#tirgul` / `#hw`
- קושי: `#easy` / `#medium` / `#hard`
- פרק: `#ch01` וכו'
- סוג: `#proof` / `#calculation` / `#conceptual` / `#counterexample`

### כשמגיעים תרגולים חדשים
1. קרא את קובץ ה-PDF
2. הגדרות ומשפטים → קישור ל-`01-lecture.md` של הפרק המתאים
3. דוגמאות מלאות → `02-tirgul.md` של הפרק המתאים
4. תרגילים חדשים → הוסף ל-`03-exercises.md` עם פתרון מלא

### כשמגיעים שיעורי בית חדשים
1. הוסף כל שאלה ל-`03-exercises.md` של הפרק הרלוונטי
2. כתוב פתרון עצמאי (זה חלק מהלמידה)
3. סמן `status: unsolved` עד שיש פתרון
4. **אל תפרסם פתרונות בשלב 2 לפני שהם מפורסמים רשמית**

### סטטוס פרק (ב-frontmatter של _index.md)
- `stub` — מבנה בלבד, אין תוכן
- `lecture-done` — תיאוריה הועברה
- `tirgul-done` — תרגול נוסף
- `exercises-done` — תרגילים עם פתרונות
- `complete` — פלאשקארדס + הכל מוכן
