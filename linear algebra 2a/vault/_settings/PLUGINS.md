# הגדרת Obsidian — תוספים נדרשים

## פתיחה ראשונה
1. פתח את Obsidian
2. בחר **"Open folder as vault"**
3. נווט אל `C:\Users\itama\code\la2-course\vault`
4. עבור אל Settings → Community Plugins → Enable community plugins
5. התקן את התוספים הבאים

---

## תוספים חובה

### 1. Obsidian Spaced Repetition
הופך את קבצי `04-flashcards.md` לחזרה מרווחת.
- חפש: `Spaced Repetition`
- לאחר התקנה: Settings → Spaced Repetition → הוסף `#flashcard` כ-tag

### 2. Dataview
מאפשר שאילתות על הקבצים — סטטוס פרקים, תרגילים שלא נפתרו.
- חפש: `Dataview`

### 3. LaTeX Suite
קיצורי מקלדת ל-LaTeX בזמן כתיבה.
- חפש: `LaTeX Suite`
- קיצורים מומלצים להגדיר בהגדרות התוסף:
  - `mk` → `$$`
  - `dm` → `$$\n\n$$`
  - `//` → `\frac{}{}`
  - `sq` → `\sqrt{}`

### 4. Excalidraw
ציור ידני לאינטואיציה גיאומטרית.
- חפש: `Excalidraw`

---

## הגדרות ידניות

### RTL ועברית
Settings → Editor → **Right-to-left** → הפעל

### מצב עריכה
Settings → Editor → **Default editing mode** → Source mode
(חשוב לעבודה תקינה עם LaTeX)

### שורת קריאה
Settings → Editor → **Readable line length** → כבה
(מתאים יותר לטקסט עברי)
